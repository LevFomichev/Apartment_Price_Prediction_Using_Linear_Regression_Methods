# Apartment_Price_Prediction_Using_Linear_Regression_Methods
## Прогнозирование стоимости жилых помещений базовыми моделями линейной регрессии

### Описание проекта

В данном мини-проекте нам будет необходимо спрогнозировать стоимость жилого помещения в местной валюте воспользовавшись двумя методами линейной регрессии. Мы должны будем оценить их результаты между собой и сделать выводы о необходимости в их дальнейшей доработке.

---

### Бизнес-задача:

Оценить необходимость в разработке более сложной модели прогноза ценообразования помещения по сравнению с «простыми» (базовыми) моделями предсказаний.

### Техническая задача для специалиста в *Data Science*:

* Провести первичный анализ имеющихся данных, подготовить признаки, которые будут использоваться при обучении и проанализировать их распределения.

* При работе с данными о ценах (целевой переменной) необходимо применить к ней логарифм в виду большого разброса цен.

* Реализовать и обучить базовую модель байесовской линейной регрессии с выбранными признаками.

* На тех же самых признаках построить базовую модель классической линейной регрессии из *sklearn* и сравнить их путём применения метрики *MAE*:

<center>

$${MAE} = \frac{1}{N} \sum_{i=1}^{N} |Actual_i - Predicted_i|$$

</center>

* Оценить полученные обоими моделями результаты и сделать выводы о необходимости в их доработке.
